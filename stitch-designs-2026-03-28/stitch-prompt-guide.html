10 prompts ready to paste into Google Stitch · Web mode · 3.0 Flash

How to Use This Guide

  1. Open stitch.withgoogle.com and make sure Web mode is selected (not App)
  2. For each prompt below, click the prompt text to copy it
  3. Paste into Stitch's input field and press Enter
  4. Recommended: Before submitting each prompt, take a screenshot of the matching page in your browser (URLs listed in each card below), then click the + button → Upload Files to attach it as a reference image
  5. After Stitch generates, click Export → save the code to stitch-designs-2026-03-28/exported-code/
  6. Check off each card as you complete it to track progress
