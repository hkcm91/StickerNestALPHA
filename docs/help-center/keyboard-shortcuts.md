# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in StickerNest. On Mac, `Cmd` is used; on Windows/Linux, substitute `Ctrl`.

## Navigation

| Shortcut | Action |
|----------|--------|
| Space + drag | Pan the canvas |
| Scroll wheel | Zoom in/out |
| `Cmd +` / `Cmd -` | Zoom in / out |
| `Cmd 0` | Reset zoom to 100% |
| `Shift + 1` | Zoom to fit (show all entities) |

## Tools

| Shortcut | Tool |
|----------|------|
| `V` | Select tool |
| `M` | Move tool |
| `B` | Pen / draw tool |
| `T` | Text tool |
| `R` | Rectangle tool |
| `E` | Ellipse tool |
| `L` | Line tool |
| `S` | Sticker placement tool |
| `W` | Pipeline wiring tool |

## Selection and Editing

| Shortcut | Action |
|----------|--------|
| Click entity | Select entity |
| `Shift` + click | Toggle in/out of multi-selection |
| Drag on empty space | Marquee (region) select |
| `Cmd + A` | Select all entities |
| `Delete` / `Backspace` | Delete selected entities |
| `Cmd + D` | Duplicate selected entities |
| `Cmd + Z` | Undo |
| `Cmd + Shift + Z` | Redo |
| `Cmd + S` | Save (canvases also auto-save) |

## Resize Modifiers

| Modifier | Behavior |
|----------|----------|
| `Shift` + drag corner | Maintain aspect ratio |
| `Alt` + drag handle | Resize from center |

## View and Mode

| Shortcut | Action |
|----------|--------|
| `P` | Toggle edit / preview mode |
| `A` | Toggle asset panel |
| `[` | Toggle left sidebar |
| `]` | Toggle right sidebar |

## Tips

- Shortcuts are context-scoped: canvas shortcuts are inactive in the Widget Lab, and vice versa
- In preview mode, most editing shortcuts are disabled since the layout is locked
- Hold a tool shortcut key temporarily to use that tool, then release to return to the previous tool
