# StickerNest Content Pipeline

Content strategy, marketing production, and brand voice for Twitter/X, TikTok, YouTube, and Reddit.

## Skills
- **content-strategy** — Content discovery, scoring, calendar management
- **marketing** — Platform-specific templates and formatting (Twitter, TikTok, YouTube, Reddit)
- **brand-voice** — StickerNest voice guidelines v1 (tone, pillars, do/don't)

## Connectors
- Notion (Build Journal, content calendar)
- Gmail (outreach drafts)
- Web search (trends, competitors)

## Depends On
- sn-core-context, sn-scribe (reads Build Journal)
